Roblox Royale High Sets#

Dear Dollie#

This is a permanent Set that contains six items. The accessories, shoes, and skirt for this set were released in July 2019 while the gloves were added later in November 2019. It is available for purchase in the shop for 120,000 Diamonds. It consists of the Dear Dollie Ruffled Skirt, Dear Dollie Ribbon Heels, Dear Dollie Laced Back Bow, Dear Dollie Tea Party Bow, Dear Dollie Laced Bodice, and the Dear Dollie Gloves.

Enchanting Heirloom#

Another permanent Set, the Enchanting Heirloom is available to purchase for 49,500 Diamonds in the shop. It consists of an Enchanting Heirloom Cage Skirt, Enchanting Heirloom Froufrous Bodice, and an Enchanting Heirloom Bouffant Sleeves. The Set was added to the game in January, 2020.

Fluttering Butterfly#

It is another one of the permanent Royale High Sets. This Set is available for purchase in the shop for 71,000 Diamonds. This fluttery set was added to the game in June 2020. It consists of the Fluttering Butterfly Bodice, Fluttering Butterfly Skirt, Fluttering Butterfly Heels, and the Fluttering Butterfly Sleeves. Players can find all the parts of this Set in the shop except for the sleeves. The sleeves  for this set can be found in a chest in the Beach House for free.

Miss Lady Rose#

Miss Lady Rose is a permanent Set that includes Miss Lady Rose (skirt), Miss Lady Rose Bow, and Miss Lady Rose Bodice. It was released in May, 2019 and can be purchased from the Shop for 76,000 Diamonds.

Darling Valentina#

This is a limited time set that was released in February 2020 as a part of the Valentines 2020 Event. The entire set costs 86,500 Diamonds but players can not purchase it currently. The Set is fully available during February only. Players looking to get the Set now would have to trade with other Royale High players with complete Sets. However, players can find the Darling Valentina Playful Puppy Ears in a chest in the Beach House. This Set contains the Darling Valentina Lace Bodice, Darling Valentina Lace Cuffs, Darling Valentina Morning & Evening Skirt, Darling Valentina Lollipop Heels, and the Darling Valentina Playful Puppy Ears.

December’s Dream#

This is a limited time Set that was a part of the Christmas 2021 Event and released on December 24, 2021. It takes 198,000 Diamonds to purchase the entire Set from the shop. However, like the Darling Valentina, this Set is also not available in the Shop currently. Players can only avail this eight item set from November 15th to January 31st. Therefore, those looking to complete this Set can either wait for it to return to the shop or trade for it with other Royale High players. It consists of the December’s Dream Cossack Hat, December’s Dream Earrings, December’s Dream Coat Jacket, December’s Dream Bodice, December’s Dream Sleeves & Gloves, December’s Dream Fur Muff, December’s Dream Dancer Skirt, and the December’s Dream Shoes. This outfit is available for both boys and girls.

Gothicutie#

This Halloween inspired outfit was released in the game on September 27, 2021. It costs 97,000 Diamonds in the shop but can only be purchased during Halloween. So, players looking to get this edgy outfit will either have to wait or trade for this Set. It consists of the Gothicutie Earrings, Gothicutie Bodice/Jacket, Gothicutie Skirt, and the Gothicutie Boots. This outfit is available for boys as well.

Midnight’s Strike Popstar#

This starry outfit first appeared as a part of the Royale High Sets during the New Year 2021 event and was released on December 30th, 2020. It made its reappearance in the game during the New Year 2022 Event. The outfits were offered as rewards for quests in the event. However, there is no telling if the outfit will make its return to the shop for another New Year Event. Players looking to get this set might have to settle for trading with other players. It consists of the Midnight’s Strike Popstar Astral Hair Bow, Midnight’s Strike Popstar Constellation Bodice, Midnight’s Strike Popstar Microphone, Midnight’s Strike Popstar Celestial Armlets, Midnight’s Strike Popstar Performance Skirt, and the Midnight’s Strike Popstar Platform Heels. Both male and female outfits are available in this set.

Mon Cheri Tea Party#

Valentines 2021 Event introduced Royale High players to this lovely Set. It was released on February 12, 2021 and is only available in February. This Set costs 116,000 Diamonds to purchase and is available for boys and girls. The sets consists of six items: Mon Cheri Tea Party Lovely Beret, Mon Cheri Tea Party Bowtie Bodice, Mon Cheri Tea Party Palace Sleeves, Mon Cheri Tea Party Clutch, Mon Cheri Tea Party Skirt, and the Mon Cheri Tea Party Platforms. This outfit is available for boys as well.

Opposites Attract#

This Set was released on February 2 as part of the Valentines 2022 Event. It costs 134,000 Diamonds but is currently unavailable in the shop. The set consists of the Opposites Attract Transformation Dress & Bodice, Opposites Attract Transformation Boots, Opposites Attract Steampunk Gloves (Sleeves), Opposites Attract Headphones, and the Mend My Heart Stitched Teddy. This Set is also available for boys.

Peppermint Princess#

This Set was originally released in November, 2021. However, additional items in this Set were introduced during the Christmas 2021 gifting Event. Players could buy the bodice and the skirt in the shop from November to January for 59,000 Diamonds. However, the sleeves were a Day 2 Christmas Gifting Event Advent Calendar reward in 2021. These can only be obtained through trading at the moment. The set consists of the Peppermint Princess Bodice, Peppermint Princess Skirt, and the Peppermint Princess Sleeves. This Set also has a male version. Learn how to get the Peppermint Princess Set in Roblox Royale High.

Princess Starfrost#

This winter themed Set was appropriately released in December 2019 during the Christmas 2019 event. It costs 125,500 Diamonds to purchase the Set but it is only available from November to January. This set contains six items. These are the Princess Starfrost Crystal Scepter, Princess Starfrost Cozy Bodice, Princess Starfrost Shooting Stars Cape, Princess Starfrost Ribbon Bonnet, Princess Starfrost Magical Skirt, and the Princess Starfrost Figure Skates.

Royale Rebel#

Introduced during the Halloween 2020 Event, this set is available for boys and girls. It is priced at 113,000 Diamonds but can only be purchased during October. The set consists of the Royale Rebel Spiked Veil, Royale Rebel Sheer Laced Gloves & Sleeves, Royale Rebel Locket Pendant Bodice, Royale Rebel Bustle Skirt, and the Royale Rebel Laced Boots.

Shadow Empress#

Another Halloween-themed Set, was released during the Halloween 2019 Event. Priced at 123,000 Diamonds, this Set can only be purchased from September 15 to November 15. It consists of the Shadow Empress Jeweled Crown, Shadow Empress Fallen Angel Sleeves, Shadow Empress Chained Bodice, Shadow Empress Jeweled Skirt, and the Shadow Empress Chained Boots.

Spring Cherry Blossom#

This Spring inspired Set was released in April 2020. It is priced at 102,500 Diamonds and is only available during April and May. The items in this Set include the Cherry Blossom Kimono Bodice, Cherry Blossom Kimono Sleeves, Cherry Blossom Kimono Skirt, Cherry Blossom Geta Sandals, Cherry Blossom Hairpins, Traditional Hand Fan, and the Traditional Back Bow. Moreover, this Spring Cheer is available for boys as well.

Summer Fantasy#

This Set was released in June 2021 and costs 164,000 Diamonds to purchase from the shop. However, it is currently offsale and will be available during the summer from June to September. This outfit has male and female versions and has six items in it. These items include the Summer Fantasy Goddess Veil, Summer Fantasy Rings and Sleeves, Summer Fantasy Beach Boardwalk Bodice, Summer Fantasy Breezy Beach Skirt, Summer Fantasy Mermaid Seashell Purse, and the Summer Fantasy Seashell Sandals.

Winter Guardian#

Released in December, this fantasy winter outfit was a part of the Christmas 2020 Event. However, it is only available from November to January now. It costs 117,000 Diamonds to purchase. The Winter Guardian Royale High Set consists of the Winter Guardian Aura Antlers, Winter Guardian Thermal Bodice/Tailcoat, Winter Guardian Enchanted Sleeves, Winter Guardian Snowstorm Skirt, and the Winter Guardian Glacier Boots.

These are all the Royale High sets available in the game. While you are here, also check out the Best Royale High Outfit Ideas.
